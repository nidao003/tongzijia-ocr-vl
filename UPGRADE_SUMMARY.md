## 🎯 使用示例

### 单文件识别（支持 PDF）
```python
from paddleocr_tool import quick_recognize

# 识别图片
text = quick_recognize("invoice.png")

# 识别 PDF（自动合并所有页面）
text = quick_recognize("document.pdf")
```

### 批量识别
```python
from paddleocr_tool import batch_recognize

# 混合格式批量处理
result = batch_recognize([
    "doc1.png",
    "doc2.pdf",
    "doc3.jpg"
])

print(f"成功: {result['successful']}/{result['total_files']}")
for r in result['results']:
    if r['success']:
        print(f"{r['filename']}: {r['text'][:50]}...")
```

### REST API
```bash
# 单文件识别
curl -X POST -F "file=@doc.pdf" http://localhost:8001/ocr

# 批量识别
curl -X POST \
  -F "files=@doc1.png" \
  -F "files=@doc2.pdf" \
  http://localhost:8001/ocr/batch
```
